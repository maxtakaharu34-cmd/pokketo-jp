# ぽけっと

大人向け お小遣い／貯金管理アプリ。ダーク基調のミニマルUI。

- 残高大表示 + 期間切替（今月／今週／全期間）の収支デルタ
- カテゴリ18種（ランチ・カフェ・サブスク・趣味・美容・etc）
- **複数の貯金目標**：旅行ファンド／家電ファンド等を並行管理
- 目標カードをタップで積立額を即追加 (-値で取崩)
- クイック金額チップ（+500/+1000/+3000/+5000/+10000）
- LocalStorage 保存
- Xシェア（残高＋上位目標の達成%）
